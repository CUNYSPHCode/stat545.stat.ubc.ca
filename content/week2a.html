Tuesday’s Class 02A: Collaboration with git and GitHub

Sep 15, 2020

From today’s class, students are anticipated to be able to:

  • use git on their own computer (locally)
  • connect between a local git repository and that repository on GitHub, using RStudio
  • make commits in git using RStudio
  • make a branch in git using RStudio or GitHub
  • use collaborative GitHub features such as Issues and pull requests.

Prepare for Class

Take about 30 minutes to get acquainted with the following before coming to class:

  • Watch STAT 545 Episode 2-A: Git and GitHub for an Organized Project (a copy is also available on Canvas)
  • Read about the basic version control workflow (without branching) at Happy git w R: Chapter 20.
  • Read the Understanding the GitHub flow to get a sense of branching.

During Class

  • 9:30 - 10:00: Vincenzo demonstrates version control and GitHub:
    • 1.5.3, 1.5.5, 4.2.3, 4.2.6
    • PR, as in https://guides.github.com/activities/hello-world/ – also 4.2.8
  • 10:00 - 11:00: Students work on their Collaborative Project Milestone 1
    • These are the most important parts of this milestone to complete in class today:
      • Make/join your project repo
      • Create a teamwork document

Useful Resources as references

  • Linking a project between GitHub and local:
    • From GitHub to local
    • From local to GitHub

Thursday’s Class 03A: Reproducible Reports with R Markdown

Sep 17, 2020

From today’s class, students are anticipated to be able to:

  • Use basic markdown features
  • Write documents in markdown on GitHub and RStudio, and render these documents to html and pdf with RStudio.
  • Choose whether html or pdf is an appropriate output
  • Style an Rmd document by editing the YAML header
  • Demonstrate at least two Rmd code chunk options

Prepare for Class

Before class, be sure to:

  • Watch STAT 545 Episode 3-A: Reproducible Reports with R Markdown (launched Tuesday at 11:00am PST)
  • Read the Official R Markdown Tutorial from the “Introduction” up to and including the “Inline Code” section.

During Class

  • 9:30 - 10:00: Vincenzo demonstrates markdown, R Markdown, and rendering/knitting files.
  • 10:00 - 11:00: Students continue working on:
    • Collaborative Milestone 1, especially formatting their documents with markdown features.
    • Worksheet 1-A, as needed.
