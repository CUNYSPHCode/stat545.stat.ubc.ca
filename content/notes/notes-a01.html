Installation

You’ll need to install some things and get set up. Most things can be found in Jenny’s “happy git” book. Let’s take some class time to struggle through this together. Need extra help? We will have an office hour on Friday, 3-4.

  • Register a github account at github.com. See Chapter 4.
  • Install or upgrade R and RStudio. See Chapter 5.
  • Install git. Chapter 6.
  • Configure git. See Chapter 7: Introduce yourself to git.
  • Install jupyterlab + IRkernel. See Rich Pauloo’s instructions, which seem to be pretty good (let us know if you find something better).
