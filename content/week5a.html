Tuesday’s Class 08A: Tidy Data

Oct 6, 2020

From today’s class, students are anticipated to be able to:

  • recognize whether a given dataset is ‘tidy’ or ‘untidy’
  • convert a dataset between the ‘long’ and ‘wide’ format, using tidyr::pivot_longer() and tidyr::pivot_wider()
  • assess which format is best suited for each type of analysis
  • deal with missing data in a tibble

Prepare for Class

  • Watch the STAT 545 Video Series: Class 08A
  • Required readings:
    • Jenny’s Intro to Tidy Data
      • Note that in tidyr 1.0.0, pivot_longer() replaced gather(), and pivot_wider() replaced spread(), which are functions you may see in older documentation
    • tidyr vignette on tidy data
    • pivot vignette with tidyr

Some useful (but not mandatory) readings are:

  • Hadley’s Paper on Tidy Data (more thorough)
  • Other new additions in `tidyr
