Tuesday’s Class 08A: Tidy Data

Oct 6, 2020

From today’s class, students are anticipated to be able to:

  • recognize whether a given dataset is ‘tidy’ or ‘untidy’
  • convert a dataset between the ‘long’ and ‘wide’ format, using tidyr::pivot_longer() and tidyr::pivot_wider()
  • assess which format is best suited for each type of analysis
  • deal with missing data in a tibble

Prepare for Class

  • Watch the STAT 545 Video Series: Class 08A (also available on canvas)
  • Read the pivot vignette with tidyr

Some useful (but not mandatory) readings are:

  • tidyr vignette on tidy data for more depth on tidy data.
  • Hadley’s Paper on Tidy Data for a formal journal article on tidy data.
  • Other new additions in tidyr v1.0.0

During Class

  • In class demo, covering:
    • pivot_longer()
    • pivot_wider()
    • univariate pivoting
    • multivariate pivoting
    • Convenience functions like unite(), separate(), expand(), complete()
  • Quiz
    • Complete the Class 08A Quiz on Canvas by Wednesday, Oct 7, 2020 at 11:00am PST.

Thursday’s Class 09A: The Model-Fitting Paradigm in R

Oct 8, 2020

From today’s class, students are anticipated to be able to:

  • make a model object in R
  • write a formula in R
  • predict on a model object with the broom::augment() and predict() functions
  • extract information from a model object using broom::tidy(), broom::glance(), and traditional means.

Prepare for Class

  • Watch the STAT 545 Video Series: Class 09A
  • Read The Model-Fitting Paradigm in R from the Tutorials section of the website.

Some useful (but not mandatory) readings are:

  • The broom vignette is useful for learning the details of broom.
  • If you want a quick intro to linear regression with R, check out this U Chicago Tutorial on model fitting in R (just the linear regression part).
  • If you’re eager to learn more about models in general, An Introduction to Statistical Learning is an approachable read (the book is freely available online)

During Class

  • In class demo, covering:
    • fitting a linear regression model using lm
    • plotting linear models using geom_smooth
    • augment()
    • tidy()
    • glance()
  • Quiz

Complete the Class 09A Quiz on Canvas by Friday, Oct 9, 2020 at 11:00am PST.
