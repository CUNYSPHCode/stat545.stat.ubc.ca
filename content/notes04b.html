Learning Objectives

This week aims to provide you with a foundation for making a dashboard using R shiny. From today’s class, students are anticipated to be able to:

  • develop basic shiny apps from scratch,
  • deploy a shiny app,
  • create interactive Shiny R Markdown documents, and
  • research more advanced shiny techniques.

And possibly:

  • Make an interactive plot with plotly
  • Make a shiny document (embed shiny features within an Rmd file)

Additional Resources

General:

  • STAT 545 Episode 4-B
  • “Mastering Shiny” by Hadley Wickham
  • The official shiny site is also very useful. It has tutorials, a gallery, and other goodies.

Specific tasks:

  • For deploying shiny apps, check out https://www.shinyapps.io/
  • For shiny documents (= R Markdown + shiny), see Chapter 19 of Yihui’s R Markdown book.
  • For testing your shiny app, check out the shinytest and reactlog packages.

Other dashboard tools besides shiny (but not in R):

  • With python: plotly dash. Check out the main website, or this Medium post introducing the tool.
  • With javascript: D3, a tremendously powerful tool with a steep learning curve (esp. if you don’t know javascript).
